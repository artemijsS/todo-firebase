## To start

1) Install all dependencies 
```bash
npm install
```

2) Fill out all .env variables (create .env.local with all variables from .env.example)
3) Start dev server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
